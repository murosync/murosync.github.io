Hardware Architecture (Concept)

Philosophy: roles, not products

Murosync explores a modular rack-mounted architecture designed to support multiple system roles built on a common platform. The goal is experimental flexibility and architecture validation.

Role configurations

  • Timing master role: global reference distribution over optical links
  • Timing distribution / repeater role: scalable cascaded timing topologies
  • Trigger generation role: deterministic trigger outputs with scheduled timing
  • Combined experimental role: integrated configuration for laboratory validation

Conceptual form factor

A 2U 19" rack device concept (“network-switch-like” ergonomics), optimized for dense optical I/O and optional trigger I/O.
